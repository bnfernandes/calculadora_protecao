Sobre o Projeto

Trabalho de Conclusão de Curso

Este site foi desenvolvido como parte do Trabalho de Conclusão de Curso (TCC) em Engenharia Elétrica, com foco em Proteção de Sistemas Elétricos. O objetivo principal é fornecer uma ferramenta prática para cálculo e parametrização das principais funções de proteção utilizadas em sistemas elétricos de potência.

O projeto utiliza como base os equipamentos da família Ingepac EF da Ingeteam, uma linha de relés de proteção amplamente utilizada no setor elétrico. As funções de proteção implementadas incluem:

  • Função 51 - Proteção de Sobrecorrente Temporizada
  • Função 67 - Proteção Direcional
  • Função 21 - Proteção de Distância
  • Função 87 - Proteção Diferencial de Transformador
  • Função REF - Proteção Diferencial de Terra Restrita
Autor

Este trabalho foi desenvolvido como requisito para obtenção do título de Engenheiro Eletricista.

O projeto busca unir conhecimentos teóricos de proteção de sistemas elétricos com aplicações práticas, facilitando o dimensionamento e parametrização de relés de proteção em situações reais.

Objetivos

Os principais objetivos deste projeto são:

  • Facilitar o cálculo e parametrização de funções de proteção em sistemas elétricos
  • Fornecer uma ferramenta educacional para estudantes de engenharia elétrica
  • Demonstrar a aplicação prática dos conceitos teóricos de proteção
  • Criar uma referência para profissionais que trabalham com equipamentos da Ingeteam

Este site está em constante desenvolvimento, com planos para adicionar mais funções de proteção e aprimorar as ferramentas de cálculo existentes.
